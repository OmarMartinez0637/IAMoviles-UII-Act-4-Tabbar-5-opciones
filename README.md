## Prompt
Lenguaje dart flutter.
Negocio Heladeria"Dairy Queen".
Usar esta paleta de colores:Para el fondo vas a usar blanco, y para lo demás vas a usar un azul y un rojo pero que sean oscuros porque la letra va a ser de color blanco y que se pueda diferenciar la letra del fondo.
Genera pagina con tabbar inferior con 5 opciones (Helados, Bebidas, Pasteles, Conos, Para llevar) con icono a su izquierda relacionado con la opcion.
En archivos de las paginas tab: Imagen en el centro subida desde la red 200x200. Usar las siguientes para cada archivo:
Helados:https://raw.githubusercontent.com/OmarMartinez0637/Imagenes-Para-DairyQueen/refs/heads/main/m%26ms.png
Bebidas:https://raw.githubusercontent.com/OmarMartinez0637/Imagenes-Para-DairyQueen/refs/heads/main/cereza.JPG
Pasteles:https://raw.githubusercontent.com/OmarMartinez0637/Imagenes-Para-DairyQueen/refs/heads/main/pastel1.PNG
Conos:https://raw.githubusercontent.com/OmarMartinez0637/Imagenes-Para-DairyQueen/refs/heads/main/cono.PNG
Para llevar:https://raw.githubusercontent.com/OmarMartinez0637/Imagenes-Para-DairyQueen/refs/heads/main/parallevar%202.PNG
Generar main y los archivos de las paginas tab y que haya navegacion entre las pantallas
